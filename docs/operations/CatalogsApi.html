getCatalog

URL

//{tenant}/catalog

HTTP Method

GET

Response Type

CatalogItem

Parameters

  • Path: tenant String

    The tenant to restrict results to.

  • Path: catalogIdentifier String

    The catalog ID

getCatalogs

URL

//{tenant}/catalogs

HTTP Method

GET

Response Type

CatalogsResponseSchema

Parameters

  • Path: tenant String

    The tenant to restrict results to.

  • Query: startIndex Integer

    The number of items to skip before starting to collect the result set.

  • Query: limit Integer

    The numbers of items to return.
